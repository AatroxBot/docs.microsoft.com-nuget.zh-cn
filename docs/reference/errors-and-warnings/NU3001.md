# <a name="nuget-error-nu3001"></a>NuGet 错误 NU3001

*NuGet 4.6.0+*

### <a name="issue"></a>问题
为参数无效[签名命令](../../tools/cli-ref-sign.md)或[验证命令](../../tools/cli-ref-verify.md)。

### <a name="solution"></a>解决方案
检查并更正提供的参数。
