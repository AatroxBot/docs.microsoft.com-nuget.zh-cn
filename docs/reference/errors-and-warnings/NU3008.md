# <a name="nuget-error-nu3008"></a>NuGet 错误 NU3008

*NuGet 4.6.0+*

<pre>The package integrity check failed.</pre>

### <a name="issue"></a>问题
包完整性检查失败，这意味着已签名的包已被篡改自正进行签名。

### <a name="solution"></a>解决方案
扫描计算机中的使用防病毒软件。 然后从计算机中删除包、 重新安装它，和重试该操作。 如果问题仍然存在，请与包源的所有者和包所有者联系。
