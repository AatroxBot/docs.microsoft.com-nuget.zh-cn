# <a name="nuget-error-nu1108"></a>NuGet 错误 NU1108

<pre>Cycle detected: A -> B -> A</pre>

### <a name="issue"></a>问题
检测到循环依赖项。

### <a name="solution"></a>解决方案
不正确; 创作包请联系软件包所有者以更正 bug。

### <a name="note"></a>说明
Visual Studio 2017 的早期版本可能会报告过此作为警告 (NU1606)。
