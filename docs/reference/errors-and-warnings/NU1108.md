# <a name="nuget-error-nu1108-previously-nu1606"></a>NuGet 错误 NU1108 (以前 NU1606)

<pre>Cycle detected: A -> B -> A</pre>

### <a name="issue"></a>问题
检测到循环依赖项。

### <a name="solution"></a>解决方案
不正确; 创作包请联系软件包所有者以更正 bug。
