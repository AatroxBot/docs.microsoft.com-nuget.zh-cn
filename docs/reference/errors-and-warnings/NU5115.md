# <a name="nuget-warning-nu5115"></a>NuGet 警告 NU5115
<pre>Description was not specified. Using 'Description'.</pre>

### <a name="issue"></a>问题

包命令未指定属性，因此改为选择默认值。


### <a name="solution"></a>解决方案

请 scpecify 提到属性作为 msbuild 属性，如果使用的项目文件的 csproj 或 nuspec 文件中。
