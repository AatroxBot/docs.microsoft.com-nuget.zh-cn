# <a name="nuget-error-nu5009"></a>NuGet 错误 NU5009
<pre>Cannot find version of msbuild.</pre>

### <a name="issue"></a>问题

NuGet 包操作找不到 msbuild。


### <a name="solution"></a>解决方案

请传递`-MsBuildPath`到 NuGet pack 命令。
