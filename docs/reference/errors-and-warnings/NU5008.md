# <a name="nuget-error-nu5008"></a>NuGet 错误 NU5008
<pre>Manifest file not found at 'F:\project\project.nuspec'</pre>

### <a name="issue"></a>问题

传递到 NuGet 包操作的 nuspec 文件不存在。


### <a name="solution"></a>解决方案

请确保传递到 nuspec 文件 NuGet pack 命令存在于磁盘上。
