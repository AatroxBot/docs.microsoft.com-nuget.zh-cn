# <a name="nuget-warning-nu1502"></a>NuGet 警告 NU1502

<pre>Unknown Compatibility Profile: 'aaa'</pre>

### <a name="issue"></a>问题
`RuntimeSupports` 包含无效的配置文件。 通常情况下，支持配置文件中找不`runtime.json`从当前的依赖项包文件。

### <a name="solution"></a>解决方案
检查`RuntimeSupports`在项目中的值。
