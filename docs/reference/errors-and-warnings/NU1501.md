# <a name="nuget-warning-nu1501"></a>NuGet 警告 NU1501

<pre>The folder 'FolderPath' does not contain a project to restore.</pre>


### <a name="issue"></a>问题
项目还原尝试操作上找不到。 

### <a name="solution"></a>解决方案
在包含项目的文件夹中运行 nuget 还原。
