# <a name="nuget-error-nu5011"></a>NuGet 错误 NU5011
<pre>Unable to extract metadata from 'project.csproj'.</pre>

### <a name="issue"></a>问题

从程序集或项目文件中提取元数据时出现运行时异常。


### <a name="solution"></a>解决方案

请创建时出现问题 https://github.com/NuGet/Home/issues 。
