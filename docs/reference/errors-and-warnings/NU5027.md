# <a name="nuget-error-nu5027"></a>NuGet 错误 NU5027
<pre>Invalid target framework for the file 'F:\project\project.dll'.</pre>

### <a name="issue"></a>问题

打包程序集不包含有效的目标 framework 元数据。


### <a name="solution"></a>解决方案

请确保要打包的程序集包含一个有效`TargetFramework`属性。
