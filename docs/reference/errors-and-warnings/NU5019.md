# <a name="nuget-error-nu5019"></a>NuGet 错误 NU5019
<pre> File not found: 'bad_file.path'</pre>

### <a name="issue"></a>问题

用于创建 NuGet 包操作包含的 nuspec 文件`files`不存在。


### <a name="solution"></a>解决方案

请确保任何`file`中的条目`files`nuspec 文件中的元素将位于指定为路径`src`。
