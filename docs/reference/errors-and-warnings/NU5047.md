# <a name="nuget-error-nu5047"></a>NuGet 错误 NU5047

<pre>The icon file size must not exceed 1 megabyte.</pre>


### <a name="issue"></a>问题 

指定为包图标的文件大于 1 mb。 NuGet 仅允许其文件大小小于 1 MB 的图标。


### <a name="solution"></a>解决方案

使用图像编辑器程序来减小包图标文件的大小。
