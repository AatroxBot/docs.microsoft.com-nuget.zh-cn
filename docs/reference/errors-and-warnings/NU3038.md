# <a name="nuget-warning-nu3038"></a>NuGet 警告 NU3038

### <a name="issue"></a>问题

验证设置需要存储库副署，但包没有存储库副署。


### <a name="solution"></a>解决方案

使用副署的存储库包的包源。
