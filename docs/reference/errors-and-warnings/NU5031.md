# <a name="nuget-error-nu5031"></a>NuGet 错误 NU5031
<pre>The license file 'LICENSE.txt' has an invalid extension. Valid options are .txt, .md or none.</pre>

### <a name="issue"></a>问题

允许仅列出的文件扩展。

### <a name="solution"></a>解决方案

使用文件扩展名从允许列表。
