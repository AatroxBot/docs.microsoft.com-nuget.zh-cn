# <a name="nuget-error-nu5020"></a>NuGet 错误 NU5020
<pre>A source file was added with an empty path.</pre>

### <a name="issue"></a>问题

已将源文件传递到`msbuild /t:pack`带有空路径的操作。


### <a name="solution"></a>解决方案

请确保所有源文件传递给`msbuild /t:pack`操作具有有效路径，并在磁盘上存在。
