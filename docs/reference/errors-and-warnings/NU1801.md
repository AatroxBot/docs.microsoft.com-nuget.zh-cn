# <a name="nuget-warning-nu1801"></a>NuGet 警告 NU1801

### <a name="issue"></a>问题
读取源时出现错误时`IgnoreFailedSources`设置为 true，将其转换为非致命警告。 这可以包含任何消息，是泛型。

### <a name="solution"></a>解决方案
编辑你的配置来指定有效的源。
