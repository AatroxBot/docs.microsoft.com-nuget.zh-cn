# <a name="nuget-error-nu5002"></a>NuGet 错误 NU5002
<pre>Please specify a nuspec, project.json, or project file to use.</pre>

### <a name="issue"></a>问题

为 NuGet 包操作不提供的任何输入的文件。


### <a name="solution"></a>解决方案

请使用 nuspec、 project.json 或项目文件路径到 pack 命令作为参数。
