# <a name="nuget-warning-nu5121"></a>NuGet 警告 NU5121
<pre> 'content' assets will not be available when the package is installed after the migration.</pre>

### <a name="issue"></a>问题

包中包含`Content`文件夹下的文件。 如果将包安装到项目的包引用类型, 这些资源将不可用。


### <a name="solution"></a>解决方案

请请求包作者修复包, 使其不再包含`Content`文件夹。 有关详细信息, 请参阅[NuGet 文档](https://docs.microsoft.com/en-us/nuget/consume-packages/migrate-packages-config-to-package-reference)。
