# <a name="nuget-error-nu3004"></a>NuGet 错误 NU3004

*NuGet 4.6.0+*

### <a name="issue"></a>问题
未签名的包提供给[nuget 验证命令](../../tools/cli-ref-verify.md)。

### <a name="solution"></a>解决方案
运行`nuget verify`使用已签名的包。 请参阅[对包进行签名](../../create-packages/Sign-a-Package.md)。
