# <a name="nuget-warning-nu5116"></a>NuGet 警告 NU5116
<pre> 'Content\sample.txt' was included in the project but doesn't exist. Skipping...</pre>

### <a name="issue"></a>问题

打包项目中指定的内容文件不存在磁盘上，正在跳过。


### <a name="solution"></a>解决方案

请确保所有项目中指定的内容文件在磁盘上存在。
