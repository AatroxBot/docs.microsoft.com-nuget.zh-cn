# <a name="nuget-error-nu5000"></a>NuGet 错误 NU5000

### <a name="issue"></a>问题

NuGet 包操作时出现问题。 我们尚未尚未分配唯一的错误代码针对该问题时，使用 NU5000。 可以帮助我们改进，请随时提出此问题与您的错误的详细信息。


### <a name="solution"></a>解决方案

检查输出窗口 （在 Visual Studio) 或控制台输出 （通过 NuGet 或 dotnet 命令行工具） 有关详细信息。
