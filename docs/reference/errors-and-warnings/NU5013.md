# <a name="nuget-error-nu5013"></a>NuGet 错误 NU5013
<pre>Failed to build 'project.csproj'</pre>

### <a name="issue"></a>问题

项目未能运行使用 NuGet 包操作时生成`-Build`选项。


### <a name="solution"></a>解决方案

请在项目中修复生成错误，然后重试。 应在控制台日志中显示失败的原因。
