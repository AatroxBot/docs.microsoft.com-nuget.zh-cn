# <a name="nuget-error-nu5003"></a>NuGet 错误 NU5003
<pre>Failed to build package because of an unsupported targetFramework value on 'System.Net'.</pre>

### <a name="issue"></a>问题

Nuspec 文件中指定的 framework 程序集不包含有效框架。


### <a name="solution"></a>解决方案

请修复中引用的程序集指定的目标框架。
