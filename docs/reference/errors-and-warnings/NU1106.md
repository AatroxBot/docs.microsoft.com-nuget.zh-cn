# <a name="nuget-error-nu1106"></a>NuGet 错误 NU1106

<pre>Unable to satisfy conflicting requests for 'PackageId': 'Conflict path' Framework: 'Target graph'</pre>

### <a name="issue"></a>问题
无法解析的依赖项约束。

### <a name="solution"></a>解决方案
编辑项目文件以指定的依赖项而不是确切版本更加开放范围。
